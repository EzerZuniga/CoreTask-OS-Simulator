# Example Test Data

Suggested manual test sequence:

1. Create process `Editor` with 256 MB and priority 2.
2. Create process `Compiler` with 1024 MB and priority 5.
3. Show CPU queue.
4. Execute FIFO once.
5. Open RAM menu and inspect usage.
6. Create files: `notes.txt`, `report.pdf`.
7. Use recursive search for PID and file names.
8. Delete one process and verify RAM release.
9. Review history and undo last action.
